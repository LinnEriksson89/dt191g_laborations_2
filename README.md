# Uppgift 2 - Blazor
Skapa en Blazor-applikation, det är valfritt vilken template man vill använda. Jag har valt att skapa ett Blazor WebAssembly-projekt och att skapa den som en PWA.

## Krav

* Webbplatsen har minst 3 undersidor
* En av undersidorna använder HttpClient för att läsa in JSON-data som skrivs ut på skärmen.
* Det är valfritt om man läser ut data från en JSON-fil eller en extern REST-tjänst.
